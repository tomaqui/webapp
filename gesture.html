수동 테스트 (연결 확인용)

※ Chrome / Edge + HTTPS 주소(예: GitHub Pages)에서 실행하세요. 카메라·블루투스 권한을 허용해야 합니다.
※ 손바닥을 카메라로 향하고, 손가락을 위로 펴면 인식이 잘 됩니다. 한 손만 인식합니다.
※ 명령은 같은 손모양이 잠깐 유지될 때 한 번 전송됩니다(떨림 방지).
