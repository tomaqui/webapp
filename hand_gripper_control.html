🤖 AI 손가락 집게 제어

엄지·검지를 벌리고 오므리면 포니봇 집게(S4)가 따라 움직입니다 · MediaPipe Hands + Web Bluetooth
블루투스
미연결
손 인식
대기
🖐️
45  DEG (집게 각도)
0° 닫힘90° 열림
수동 테스트 (카메라 없이)
사용 방법
1) 포니봇에 ai_gripper_ble.py 실행 → OLED에 BLE: WAITING 표시
2) 블루투스 연결 버튼 → 목록에서 ESP-ARM 선택
3) 카메라 시작 → 손을 화면에 보이고 엄지·검지를 벌렸다 오므리기
⚠️ Web Bluetooth·카메라는 HTTPS에서만 동작합니다(GitHub Pages 권장). Chrome/Edge(데스크톱·안드로이드) 지원, iOS Safari 미지원.
🔧 집게가 반대로 움직이면 포니봇 코드에서 보내는 값을 90 - angle로 바꾸거나, 아래 INVERT를 true로 하세요.
