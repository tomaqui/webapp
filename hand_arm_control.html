사용 순서
1) 포니봇에서 ai_arm_ble.py 실행 → OLED BLE WAITING
2) 블루투스 연결 → ESP-ARM 선택 → 카메라 시작
3) 손을 화면 가운데에 두고, 좌우로 옮겨 회전 / 위아래로 옮겨 상하 / 엄지검지로 집게 조작
4) 특정 관절만 정밀하게 맞추고 싶으면 수동을 체크하고 슬라이더로 조정
⚠️ HTTPS(예: GitHub Pages)에서만 동작, Chrome/Edge(데스크톱·안드로이드) 지원 · iOS Safari 미지원
